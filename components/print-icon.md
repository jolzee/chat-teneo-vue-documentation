# Print Icon

## Output Parameter

{% hint style="info" %}
This output parameter must be defined on the greeting response at the beginning of a session. The icons can be any MDI icon name. 
{% endhint %}

```text
print = {
  "icon"    : "printer",
  "color"   : "white"
}
```
